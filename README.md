# Shopping_List_App

Wykorzystano w projekcie bazę danych Firebase Realtime Database.

Instrukcja obsługi:

Po uruchomieniu aplikacji MyApp widoczne jest główne okno do nawigacji. Z poziomu tego widoku możliwe jest dodanie nowego produktu jak również przejrzenie tych istniejących. 

Kliknięcie w przycisk "Store list" przeniesie nas do nowego okna, w którym można dodać nowy sklep do listy jak również przejrzeć istniejące sklepy. Naciśnięcie checkboxa "Add to favourites" umożliwia dodanie wybranego sklepu do listy ulubionych. Należy wspomnieć, że przy dodawaniu nowego sklepu zapisywana jest aktualna lokalizacja naszego urządzenia.

Kliknięcie w przycisk "Map" przeniesie nas do nowego okna, w którym widoczna jest mapa ze wszystkimi zapisanymi na liście sklepami. Ulubione sklepy mają ustawione markery na kolor pomarańczowy. Dodatkowo po kliknięciu w marker jest informacja czy dany sklep należy do ulubionych czy też nie.
